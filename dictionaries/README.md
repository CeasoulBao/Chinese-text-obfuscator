# 社区词典

这里存放网友通过 Pull Request 共享的离线 JSON 词典包。

## 使用

1. 下载所需的 `.json` 文件。
2. 在字隙的“文本混淆”标签页展开词典替换模块。
3. 点击“导入 JSON”，检查名称和有效词条数量。
4. 点击“合并这个词典”。

应用不会联网加载此目录；用户必须主动下载和导入文件。

## 格式

词典必须符合 [`schema.json`](schema.json)。可从
[`example.json`](example.json) 复制一份开始编辑。

同一个 `source` 可以包含多个 `replacements`。应用会使用当前随机种子从候选词中选择，重复候选会在导入时去除。

## 文件命名

- 使用小写英文、数字和短横线，例如 `northeast-places.json`。
- 一个文件尽量聚焦一个主题。
- UTF-8 编码，文件不得超过 1 MB。
- 不要提交个人身份信息、联系方式或任何可执行内容。

## 许可证

每个词典必须声明允许再分发和修改的许可证。推荐 `CC0-1.0` 或 `CC-BY-4.0`。贡献者应确保自己有权提交词典内容。
