# 贡献指南

感谢帮助改进字隙。

## 提交社区词典

1. Fork 仓库并创建分支。
2. 在 `dictionaries/` 中新增一个 JSON 文件。
3. 按 `dictionaries/schema.json` 校验格式。
4. 删除重复、空白和明显错误的候选词。
5. 在 Pull Request 中说明词典主题、来源和许可证。

维护者会检查：

- JSON 格式和 1 MB 大小限制。
- 是否包含至少一个有效候选词。
- 是否存在恶意 HTML、脚本、个人信息或无关内容。
- 声明的许可证是否允许开源分发。

## 提交代码

```bash
npm install
npm run test:run
npm run build
```

请为行为变化补充测试，并保持所有文字处理在浏览器本地完成。不要加入分析脚本、远程字体、自动联网词典或原文上传功能。

## 安全边界

贡献内容不得声称能够保证绕过 AI、OCR、平台审核或账号限制。字隙是可审计的文本转换工具，不是匿名通信系统。
